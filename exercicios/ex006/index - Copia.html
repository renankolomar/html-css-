Tipos de formatação

Negrito / Destaque

Podemos destacar um texto com a tag B (Não-semântica).

Devemos destacar um texto com a tag STRONG (Semântica).

Itálico / Ênfase

Podemos usar a tag I para deixar o texto em itálico/ênfase (Não-semântica).

Devemos usar a tag EM para deixar o texto em itálico/ênfase (Semântica).

Texto marcado

Para marcar um texto usamos a tag MARK.

Você pode mudar a cor de todos os marks aplicando um estilo em CSS.

Texto Grande e Pequeno

O texto grande pode ser definido em CSS já que não tem valor semântico, portanto a tag BIG que apesar de ainda funcionar, se tornou obsoleta.

Você pode usar a tag small para deixar textos menores, ou seja, com menos relevância.

Texto excluído

O texto excluído indica um texto que se deve ler mas não deve ser considerado.

Texto inserido

O texto inserido fica sublinhado indicando que foi inserido.

Sobrescrito

O texto fica sobrescrito como em x12 + 5.

Subscrito

O texto fica subescrito come em H2O.

Monoespaçado

usando a tag CODE o texto fica Monoespaçado

Pré formatado

usando a tag PRE o texto fica da mesma maneira que você digitar ele no código,mantendo as quebras de linha, por exemplo:

        
    num = int(input('escreva um número'))
    if num % 2 == 0:
        print(f'o número é par')
    else:
        print(f'o número é ímpar')
    print(f'FIM DO PROGRAMA')

Citação

Usa-se a tag q para fazer uma citação. exemplo de citação

Citação longa

indica uma citação deslocada em parágrafro próprio, geralmente longa ocupando um bloco inteiro de espaço.

Abreviações

Usa-se a tag ABBR para mostrar o significado de uma abreviação como em HTML e CSS.

Inverter o texto

A tag BDO simplesmente pode inverter o texto da esquerda para a direita ou da direita para a esquerda. Receba!
